# Telegram Wallet Tracker Bot

## Описание

Простой Telegram-бот для учёта доходов и расходов.  
Позволяет добавлять доходы, расходы, смотреть баланс, отчёты и категории.  
Имеет связь с администратором через команду /support.

## Команды

/start — Приветствие и список команд  
/help — Список команд и помощь  
/add_income сумма источник — Добавить доход  
/add_expense сумма категория комментарий — Добавить расход  
/balance — Показать текущий баланс  
/report — Показать статистику и отчёт  
/categories — Показать расходы по категориям  
/support текст — Отправить сообщение администратору

## Требования

- Python 3.7+  
- Стандартная библиотека (без сторонних модулей)

## Запуск

1. Получите токен от @BotFather в Telegram  
2. Установите переменную окружения BOT_TOKEN с вашим токеном (или через Railway Variables)  
3. Запустите main.py

```bash
python main.py
